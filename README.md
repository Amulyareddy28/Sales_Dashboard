# Sales Dashboard Analysis
## 📌 Project Title
Sales Dashboard Analysis using Power BI

## 🎯 Objective
The objective of this project is to analyze sales performance data and create an interactive dashboard that helps in understanding:
- Overall sales performance
- Profit trends
- Customer behavior
- Regional performance
- Product category analysis

The dashboard helps users make data-driven business decisions through visual insights.

## 🛠️ Technologies Used
- Microsoft Power BI
- Microsoft Excel / CSV Dataset
- Data Visualization Techniques
- DAX Functions
- Power Query

## 📂 Steps Included
1. Collected and imported the sales dataset into Power BI
2. Cleaned and transformed data using Power Query
3. Created relationships between tables
4. Added KPI cards for important metrics
5. Used slicers for interactive filtering
6. Designed charts and graphs for better visualization
7. Created dashboard layout and formatting
8. Generated insights from the dashboard

## 📊 Analysis Done
- Sales Analysis
- Profit Analysis
- Region-wise Sales Analysis
- Category-wise Performance
- Monthly Sales Trends
- Customer Segment Analysis
- Top Performing Products Analysis

## 💡 Key Insights
- Identified top-performing products and categories
- Observed sales trends across different months
- Found regions generating highest revenue
- Compared profit and sales performance
- Identified areas with lower profitability
- Improved understanding of customer purchasing behavior

## ✅ Conclusion
This dashboard provides a clear and interactive view of sales data. It helps businesses monitor performance, identify trends, and make better strategic decisions using data visualization techniques.
